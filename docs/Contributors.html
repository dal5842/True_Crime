My contributions to this project range over several responsibilities. In order to transform the raw text files of the "Cold Case" transcripts to clean, source text files our team could work with, I developed an XSLT that applied XML markup to all 23 text files. Integrated within this XSLT was Regex, this facilitated the markup process to tackle the several inconsistencies within the transcripts, to lay a solid foundation for the next phase of our project, text analysis! After establishing our research goals for our text analysis, more XSLTs were developed (with the help of Dr.B) to assist us with this process. Using the outputs of an XSLT that created text files consisting of any music within the episodes, I based my text analysis on finding patterns of theme within the word frequency of the music lyrics within these text files in hopes they correlate to the theme of it's respective episode. To do this, I used Jupyter Notebook to create an SVG of a bar graph that displayed the most frequently used verbs in all of the music text files. Then, to develop a means of text analysis with more intricacy, I used Jupyter Notebook to produce TSV files to input within Cytoscape. This outputted a network that displays the most frequently mentioned verbs within each episode. In order to display these developments, I created a soundtrack page on our website that provides context for users to understand my research goals. Additionally, I worked with Emily to make our Cytoscape page fully interactive with additional descriptive elements. To do this, I used Jupyter Notebook to output Java Script files to embed within the HTML file for this Cytoscape webpage.
